* Orderless Completion Qualifiers
These modifiers work inside any Orderless-powered completion (Vertico, Corfu):

| Qualifier | Meaning       | Example           |
|-----------+---------------+-------------------|
| ~!~       | Exclude       | ~!test~           |
| ~,~       | Initialism    | ~,rf~ → ~read-file~ |
| ~=~       | Literal match | ~=exact~          |
| ~~~       | Flex match    | ~~abc~            |
| ~%~       | Char-fold     | ~%ü~ matches ~u~  |
